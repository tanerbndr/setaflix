🔥 Firebase Kurulumu

Gerçek zamanlı senkronizasyon için Firebase config bilgilerini gir. Sadece bir kere yapman yeterli, tarayıcıya kaydedilir.

1
console.firebase.google.com adresine git → Proje oluştur
2
Sol menü → Build → Realtime Database → Create database → Test mode seç
3
Proje ayarları (⚙️) → General → Web uygulaması ekle → firebaseConfig kopyala

Config nesnesindeki değerleri aşağıya gir:

SeTaFlix
Arkadaşınla aynı anda, senkronize izle
YouTube
m3u8 / MP4
Gerçek zamanlı sync
Sohbet
Telefon uyumlu
